Before you run the code, please ensure that you have installed:
	scipy
	numpy
	sklearn
	pandas
	matplotlib
in Python.
The code is run by navigating to this folder on a terminal and running the command:
	python clustering_final.py
On running the code, it sequentially produces the required graphs.
Note that once the matplotlib window opens to show a graph, one has to
close it so that the code and resume running and produce the next graph.
